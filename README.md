## **Procedure**

### **1. Understanding Prompt Patterns**

Several prompt engineering patterns are mentioned:

| Prompt Pattern          | Description                                     | Purpose                                                  |
| ----------------------- | ----------------------------------------------- | -------------------------------------------------------- |
|   Query Decomposition   | Breaking complex tasks into smaller queries     | Improves accuracy and focus                              |
|   Decision Making       | Asking AI to choose between multiple options    | Helps refine direction and content flow                  |
|   Answer Engineering    | Instructing structure and style of response     | Ensures consistency and clarity                          |
|   Fact Checklist        | Prompting AI to verify factual accuracy         | Enhances reliability for reports                         |
|   Tail Generation       | Extending narrative or analysis logically       | Adds depth and continuity                                |
|   Menu Actions          | Offering the AI multiple choices to select from | Increases creative control                               |
|   Semantic Filtering    | Applying style or tone filters                  | Ensures tone consistency (formal, educational, creative) |

## **5. Discussion**

This experiment demonstrates how **prompt sophistication directly influences content quality**:

* Simple prompts produce generic results.
* Structured prompts using decomposition or filtering yield logical and stylistically consistent outputs.
* Creative tasks benefit from decision-making and tail generation, while analytical tasks improve with answer engineering and fact-checking.

AI tools like ChatGPT respond best when **context, role, and format are specified** clearly.
